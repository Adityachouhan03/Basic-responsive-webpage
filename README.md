# HobbyCue UI Recreation

This project is a recreation of the HobbyCue UI, built with React and Vite. It features a responsive design with a modern and clean user interface.

## Features

- **Header:** Includes a logo, search bar, navigation, and user actions.
- **Hero Section:** An engaging introduction to the platform.
- **Login/Signup Form:** A modern form with social authentication options.
- **Additional Sections:** Includes "Add your own," "Testimonials," and a comprehensive footer.
- **Responsive Design:** The layout adapts to different screen sizes, from mobile to desktop.

## Tech Stack

- **React:** A JavaScript library for building user interfaces.
- **Vite:** A fast build tool and development server.
- **CSS:** Custom styling for all components.

## How to Run

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. **Open the browser:**
   The application will be available at `http://localhost:5173/`.
